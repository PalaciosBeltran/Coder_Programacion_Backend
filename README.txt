# MongoDB / CRUD en MongoDB 

***

Descripción

> Repositorio correspondiente al desafío entregable número 8 de la clase 18
del curso Backend de CoderHouse.
> Ejercicios de persistencia de datos haciendo uso de MongoDB.

***

Requisitos

> Terminal (cmd, powershell, entre otras)
> MongoDB
> Mongoshell

***

Ejecución

> Ejecutar Mongoshell
>> Ejecutar en una terminal el servidor de Mongo mediante el comando "mongod --dbpath %RUTA_DATA_MONGO%".
>> Ejecutar en una terminal diferente el usuario mediante el comando "mongosh".
>> En caso de tener configurado el archivo mongod.cfg solo ejecutar "mongod -f %RUTA_ARCHIVOCFG_MONGO%".

> Seguir las secuencias del archivo index.sh.
